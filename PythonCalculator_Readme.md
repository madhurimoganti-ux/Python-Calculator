Python Calculator Description:
A simple and user-friendly calculator built with Python. This project performs basic arithmetic operations and is a great example of beginner-level Python concepts such as functions, conditionals, and loops.
📌 Features
    ➕ Addition
    ➖ Subtraction
    ✖️ Multiplication
    ➗ Division
    ⚠️ Error handling (e.g., division by zero)
    🔁 Continuous calculations (loop support)
Technologies Used
    Python 3
    Standard Python libraries (no external dependencies)
